Admob-for-flash

admob ane lib for flash iOS app and flash android app.it enable air application load native Advertising。show actionscript 3 load advertising

Download this project as a .zip file Download this project as a tar.gz file

Welcome to admob for flash.

admob for flash is a free ane lib for flash developer.it enable flash mobile application load admob advertising. it is easy to use.

user can show all type of banner ad and full screen(interstitial advertising) ad of admob user can show landscope and portrait advertising in actionscript 3 mobile application user can set postion of ad

lib for android can been download from http://code.google.com/p/flash-for-mobile/

Requirements

flash air sdk 3.5 or higher,

iPhone or iPad or android devices

mark

this lib do not work on pc and Simulator，but you can pack and debug on pc.and show a well on device

download

http://flash-air-admob-ane-for-ios.googlecode.com/files/admob_ios_android_ane6.3.zip

useage

you can use this lib on pc and mac ,build for android or ios air application with flash bulder or flash cs or flash develop.just do like this

var admob:Admob=Admob.getInstance();
if (admob.isSupported)
{
    admob.createADView(AdSize.BANNER, "a1514bcdferve3437"); //create a banner ad view.this init the view 
    admob.addToStage(0, 0); // ad to displaylist position 0,0
    admob.load(false); // send a ad request.  
}

for android. you need add in application-app.xml

useage detail

more function
1. handler  ad event  like this.
admob.dispatcher.addEventListener(AdEvent.onReceiveAd,onAdEvent);
2. get ad size info
admob.getAdSize()
3.get screen size info
admob.getScreenSize()
4. get mac address or device id of device
admob.getMacAddress()
